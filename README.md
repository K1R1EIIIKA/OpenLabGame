# «Snowed In» game for OpenLabGame GameJam
## **Link**: https://k1r1eiiika.itch.io/snowedin
### Metroidvania, where a little mouse has to learn a terrible secret and explore the big world!
### In the game you will have to find items, complete quests, communicate with NPCs and find secret places with valuable rewards!
